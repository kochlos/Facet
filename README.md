# 🌸 Facet

Превращает фотографию в мозаику из плиток атласа. Каждый пиксель исходника заменяется тайлом с наиболее близким цветом. Результат скачивается как PNG или открывается в интерактивном 3D.

## Как пользоваться

**1. Загрузи исходное фото (секция 01)**
Подходит JPEG, PNG и другие форматы. Предпросмотр появится сразу.

**2. Загрузи атлас (секция 02)**
Атлас — лист с тайлами 100×100 px, каждый тайл станет «пикселем» мозаики. По умолчанию загружается встроенный атлас, можно заменить своим.

**3. Опционально: добавь второй атлас (03) или атлас камней (04)**
Второй атлас позволяет миксовать два набора тайлов через слайдер **Secondary mix**. Атлас камней (с прозрачным фоном) накладывает декоративный слой через слайдер **Stone overlay**.

**4. Обрежь края если нужно (секция 05)**
Поля `↑ ↓ ← →` задают обрезку в пикселях до генерации. Кнопка **↺ Reset** сбрасывает всё в ноль.

**5. Настрой параметры (секция 06) и нажми «Generate»**
Secondary mix (0% = только первый атлас, 100% = только второй). Stone overlay — доля тайлов, заменённых камнями, режим **⠿ Scatter** — случайное распределение.

---

## Атласы

| Секция | Назначение |
|--------|------------|
| 02 — Primary Atlas | Основной набор тайлов. Обязателен. |
| 03 — Secondary Atlas | Дополнительный набор. Активирует слайдер Secondary mix. |
| 04 — Stones Atlas | Декоративный слой с прозрачным фоном (PNG). Активирует Stone overlay. |

---

## После генерации

- **↓ Save PNG** — скачать результат
- **✏ Edit** — редактор: кликни тайл на мозаике, затем нужный тайл в атласе внизу. Есть заливка **🪣 Fill**, удаление **⌫ Delete**, отмена **↩ Undo** (или `Ctrl+Z`) и масштаб
- **⬡ 3D Preview** — мозаика как рельефная плоскость. Тащи мышью для вращения, колесо — зум, кнопка ☀/☾ — направленный свет. Нормал-карта строится автоматически

## Разместить онлайн

**Netlify** — самый быстрый способ: перетащи `index.html` на [netlify.com/drop](https://netlify.com/drop).

**GitHub Pages** — создай репозиторий, загрузи файл как `index.html`, включи Pages в Settings.

---

## Технически

Всё работает в браузере — никакие файлы никуда не отправляются.

Зависимости: Three.js (подгружается по CDN только при открытии 3D).
