🌸 Facet
Превращает пиксельное изображение в мозаику из плиток атласа. Каждый пиксель исходника заменяется тайлом с наиболее близким цветом. Результат скачивается как PNG или открывается в интерактивном 3D.

Как пользоваться

1. Загрузи исходное фото (секция 01)
   Подходит JPEG, PNG и другие форматы. Предпросмотр появится сразу.

2. Загрузи атлас (секция 02)
   Атлас — лист с тайлами 100×100 px, каждый тайл станет «пикселем» мозаики. По умолчанию загружается встроенный атлас, можно заменить своим.

3. Опционально: добавь второй атлас (03) или атлас камней (04)
   Второй атлас позволяет миксовать два набора тайлов через слайдер Secondary mix. Атлас камней (с прозрачным фоном) накладывает декоративный слой через слайдер Stone overlay.

4. Обрежь края если нужно (секция 05)
   Поля ↑ ↓ ← → задают обрезку в пикселях до генерации. Кнопка ↺ Reset сбрасывает всё в ноль.

5. Настрой параметры (секция 06) и нажми ▶ Generate


Атласы

Секция	Назначение
02 — Primary Atlas	Основной набор тайлов. Обязателен.
03 — Secondary Atlas	Дополнительный набор. Активирует слайдер Secondary mix (0% = только первый, 100% = только второй).
04 — Stones Atlas	Декоративный слой с прозрачным фоном (PNG). Активирует слайдер Stone overlay и режим ⠿ Scatter.


После генерации

↓ Save PNG — скачать результат.
✏ Edit — открыть редактор, где можно вручную заменять тайлы: кликни тайл на мозаике, затем нужный тайл в атласе внизу. Есть заливка 🪣, удаление ⌫, отмена ↩ и масштаб.
⬡ 3D Preview — открыть мозаику как рельефную плоскость. Тащи мышью для вращения, колесо — зум, кнопка ☀/☾ включает направленный свет. Нормал-карта строится автоматически.


Технически

Всё работает локально в браузере — никакие файлы никуда не отправляются.
Зависимости: Three.js (подгружается по CDN только при открытии 3D).
